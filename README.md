# Extra Themed Icons

## Description
This Magisk Module adds more icons either lines or filled (you can choose) than a normal Pixel Launcher.

## Requirements
- ROOT

## Compatibility
- Android 12
- Android 12.1/12L
- Android 13

## Notes
- I recommend disabling and enabling the thematic icons so that the module is well applied.

## Tested on Device
- Motorola One Fusion
